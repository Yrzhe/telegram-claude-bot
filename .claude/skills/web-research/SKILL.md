# Web Research Skill

帮助用户搜索和获取网络上的最新信息。

## 使用场景

- 用户询问最新新闻或事件
- 用户需要查找某个话题的信息
- 用户需要了解产品、公司、人物的信息
- 用户需要获取实时数据或最新资讯
- 用户需要查找技术文档或教程

## 执行步骤

1. 分析用户需求，确定搜索关键词
2. 使用 WebSearch 工具搜索相关信息
3. 如果需要详细内容，使用 WebFetch 获取网页详情
4. 整理搜索结果，提取关键信息
5. 用简洁清晰的方式向用户汇报结果
6. 如有需要，将研究结果保存到 analysis/ 文件夹

## 注意事项

- 搜索关键词要精准，使用多个相关词组合
- 优先选择权威来源（官方网站、知名媒体）
- 注明信息来源和时间
- 如果信息可能过时，提醒用户核实
- 对于敏感话题，提供多角度的信息
